# Minimal Workspace Example

这是一个最小工作区示例，用来展示 CodexWorkspace Methodology 落地后的基本形态。

## 文件结构

```text
minimal-workspace
├── AGENTS.md
├── codexworkspace-console.md
├── workspace-architecture-and-naming.md
├── new-project-sop.md
├── projects-index.md
├── codex-issue-log.md
├── codex-skills-inventory.md
├── projects/
├── shared/
├── archives/
└── temp/
```

## 使用方式

1. 复制本目录。
2. 将示例文件中的 `ExampleWorkspace` 替换成你的工作区路径。
3. 根据你的命名规则、项目类型和 skills 来源调整模板。
4. 让 Codex 先读取 `AGENTS.md` 和 `codexworkspace-console.md`。
